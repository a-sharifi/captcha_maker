# Captcha_maker

<p align="center"><img width=100% src="https://github.com/amirsharifi74/captcha_maker/blob/master/captcha/2.png"></p>

> **A package to make captcha from your txt.**

- this package is suitable for left to right languages .
 
- Just you can use with python2.

**Just fill your words in the sample.txt and run it**

```
python LTR.py -f ~/YOUR_FONT/*.ttf -p ~/YOUR_SAMPLE_PATH/*.txt
```
